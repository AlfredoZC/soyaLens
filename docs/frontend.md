# Documentación del Frontend (SoyaLens)

Este archivo mantiene un registro vivo de la arquitectura, componentes UI y decisiones de diseño del Frontend (Streamlit/Web).
**Los agentes deben actualizar este documento automáticamente al realizar cambios.**

## Arquitectura Base
- **Framework:** Streamlit (o Web Custom)
- **Diseño:** Alta fidelidad, responsive, focalizado en agilidad para captura de datos y renderizado de certificado.
- **Comunicación:** Integración mediante peticiones HTTP a los endpoints de la API en `backend/`.
